# 主要領域セクション用画像

このディレクトリには、作業請負事業の主要領域カード用背景画像を配置します。

## 必要な画像

以下の3枚の画像が必要です:

### 1. warehouse.jpg (倉庫業務)
- **推奨サイズ**: 800x600px以上
- **アスペクト比**: 4:3 または 16:9
- **テーマ**: 倉庫、物流センター、在庫管理、フォークリフト
- **推奨サイト**:
  - Unsplash: https://unsplash.com/s/photos/warehouse
  - Pexels: https://www.pexels.com/search/warehouse/
  - Pixabay: https://pixabay.com/images/search/warehouse/

**推奨画像例**:
- https://unsplash.com/photos/a-large-warehouse-filled-with-lots-of-boxes-ydbNa5YY1NY
- https://unsplash.com/photos/white-boxes-on-brown-wooden-pallet-4UrqFnPJp0w
- https://unsplash.com/photos/person-in-orange-and-white-striped-polo-shirt-sitting-on-chair-FlPc9_VocJ4

### 2. inspection.jpg (検品・梱包)

- **アスペクト比**: 4:3 または 16:9
- **テーマ**: 検品作業、梱包作業、品質管理、商品チェック

## ダウンロード方法

1. 上記のリンクから気に入った画像を選ぶ
2. 「Download」または「Free Download」ボタンをクリック
3. ダウンロードした画像を以下のファイル名にリネーム:
   - `warehouse.jpg`
   - `inspection.jpg`
   - `office.jpg`
4. このディレクトリ(`site/images/areas/`)に配置

## 画像の最適化

画像サイズが大きい場合は、以下のツールで最適化することをおすすめします:
- TinyPNG: https://tinypng.com/
- ImageOptim (Mac): https://imageoptim.com/
- Squoosh: https://squoosh.app/

## 使用方法

これらの画像は**背景画像として薄く表示**されます:
- 透明度: 15%（ホバー時25%）
- カードの下部からグラデーションオーバーレイがかかり、テキストが読みやすくなります
- 画像は `object-fit: cover` で表示され、カード全体を覆います

## ライセンスについて

- **Unsplash**: 商用利用可能、クレジット表記不要（推奨）
- **Pexels**: 商用利用可能、クレジット表記不要
- **Pixabay**: 商用利用可能、クレジット表記不要

ただし、各サイトの利用規約を必ず確認してください。

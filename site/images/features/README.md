# 特徴セクション用画像

このディレクトリには、事業紹介ページの特徴カード用画像を配置します。

## 必要な画像

以下の3枚の画像が必要です:

### 1. ai-analysis.jpg (AI分析)
- **推奨サイズ**: 800x600px以上
- **アスペクト比**: 4:3 または 16:9
- **テーマ**: AI、データ分析、グラフ、ダッシュボード、統計
- **推奨サイト**:
  - Unsplash: https://unsplash.com/s/photos/ai-analysis
  - Pexels: https://www.pexels.com/search/data%20analysis/
  - Pixabay: https://pixabay.com/images/search/artificial%20intelligence/

**推奨画像例**:
- https://unsplash.com/photos/turned-on-monitoring-screen-hpjSkU2UYSU
- https://unsplash.com/photos/person-using-macbook-pro-on-person-s-lap-npxXWgQ33ZQ
- https://unsplash.com/photos/macbook-pro-showing-programming-language-1SAnrIxw5OY

### 2. track-record.jpg (豊富な実績)

## ダウンロード方法

1. 上記のリンクから気に入った画像を選ぶ
2. 「Download」または「Free Download」ボタンをクリック
3. ダウンロードした画像を以下のファイル名にリネーム:
   - `ai-analysis.jpg`
   - `track-record.jpg`
   - `quality-control.jpg`
4. このディレクトリ(`site/images/features/`)に配置

## 画像の最適化

画像サイズが大きい場合は、以下のツールで最適化することをおすすめします:
- TinyPNG: https://tinypng.com/
- ImageOptim (Mac): https://imageoptim.com/
- Squoosh: https://squoosh.app/

## ライセンスについて

- **Unsplash**: 商用利用可能、クレジット表記不要（推奨）
- **Pexels**: 商用利用可能、クレジット表記不要
- **Pixabay**: 商用利用可能、クレジット表記不要

ただし、各サイトの利用規約を必ず確認してください。

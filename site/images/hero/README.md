## ダウンロード方法

1. 上記のリンクから気に入った画像を選ぶ
2. 「Download」または「Free Download」ボタンをクリック
3. ダウンロードした画像を以下のファイル名にリネーム:
   - `warehouse.jpg`
   - `ai-data.jpg`
   - `online-learning.jpg`
4. このディレクトリ(`site/images/hero/`)に配置

## 一時的な対応（URLで直接参照）

画像をダウンロードする前に、Unsplashの画像を直接URLで参照することもできます。
HTMLファイルの画像パスを以下のように変更:

```html
<div class="hero-slide active" style="background-image: url('https://images.unsplash.com/photo-1553406830-ef2513450d76?w=1920&q=80')"></div>
<div class="hero-slide" style="background-image: url('https://images.unsplash.com/photo-1551288049-bebda4e38f71?w=1920&q=80')"></div>
<div class="hero-slide" style="background-image: url('https://images.unsplash.com/photo-1588196749597-9ff075ee6b5b?w=1920&q=80')"></div>
```

## ライセンスについて

- **Unsplash**: 商用利用可能、クレジット表記不要（推奨）
- **Pexels**: 商用利用可能、クレジット表記不要
- **Pixabay**: 商用利用可能、クレジット表記不要

ただし、各サイトの利用規約を必ず確認してください。
